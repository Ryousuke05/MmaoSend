# 猫猫传送 (MmaoSend)

![MmaoSend](https://raw.githubusercontent.com/Ryousuke05/MmaoSend/main/static/icon.ico)  
猫猫传送是一个隐私、~~稳定~~、便携、跨平台的开源局域网文件传输工具喵。无需在手机/平板端安装额外应用即可使用。  

> [!NOTE]
> 本项目使用AI技术辅助创作

## 功能特点
- 主要功能：支持传输各种类型的文件，文字消息喵
- 安全稳定：文件局域网传输，不经过公网服务器喵
- 跨平台：支持**电脑**与手机/平板/手表等带浏览器的设备直接传输，无需安装额外应用喵
- 无需安装：便携式设计，只需启动程序本体即可使用，无需安装，可以放在U盘里，到哪都能用
- 体积小巧：程序本体仅15-25MB左右

## 快速上手
> [!TIP]
> **如果你不想折腾，可以在[这里](https://github.com/Ryousuke05/MmaoSend/releases)下载发行版喵 ，点开就能用～～**  
（使用发行版可忽略下方环境安装/启动）

### 环境安装与启动
1. clone本项目`git clone https://github.com/Ryousuke05/MmaoSend.git`
2. 安装环境：  
双击文件夹内的install.bat会自动安装Python（如果没有）和环境
3. 启动：  
双击start.bat启动服务，会自动弹出浏览器页面

## 故障排除
Q: 软件启动了，电脑上显示了二维码，但是手机扫了打不开怎么办  
A: 1. 确保手机和电脑已连接到同一网络下喵 2. 可以在电脑网页上选择另外一个IP，然后重新在手机端扫码喵 3. 检查防火墙设置 4. 重新启动电脑再试

Q: 我双击了exe程序，但是完全没有反应怎么办  
A: 请确保你的电脑已设置一个默认浏览器，检查后台是否有名为猫猫传送.exe的进程运行，如果依然不能解决，请提交issue  

Q: 我不小心把浏览器窗口关掉了怎么办  
A: 程序检测到浏览器页面关闭后会在8秒后停止运行，请等待8秒左右再重新打开。
